## BLE Advertisment

- [W16LU] `manufacturerID`
- [W16LU] `deviceSKU`
- [W8] `version.major`
- [W8] `version.minor`
- [W8] `version.patch`
- [BUF6] `deviceId`
- [W16LU] `position`
